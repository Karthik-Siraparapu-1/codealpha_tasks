# Credit Risk Decision Engine

## Overview
A machine learning system to predict loan approval probability and classify applicant risk.

## Features
- Multi-model evaluation
- Cross-validation
- OneHotEncoding + StandardScaler
- ROC-AUC scoring
- Feature importance analysis
- Risk categorization
- Manual review decision engine
- Applicant report generation

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting

## Final Results
- Best Model: Random Forest
- Accuracy: 92.99%
- ROC-AUC: 0.9745

## Outputs
- best_credit_model.pkl
- feature_importance.png
- model_comparison.csv
- classification_report.txt
- applicant_report.txt

## Real-world Applications
- Loan underwriting
- Credit risk assessment
- Fintech lending systems
- Banking decision support

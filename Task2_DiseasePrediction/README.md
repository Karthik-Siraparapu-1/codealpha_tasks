# Disease Prediction System

A machine learning system for predicting heart disease risk using patient medical records.

## Features

* Multi-model comparison
* Logistic Regression
* Random Forest
* SVM
* ROC-AUC evaluation
* Confusion matrix
* Feature importance analysis
* Risk level classification
* Prediction exports
* Model saving

## Tech Stack

Python
Pandas
Scikit-learn
Matplotlib
Seaborn
Joblib

## Dataset

Heart Disease Dataset

## Outputs

* Confusion Matrix
* Model Comparison Graph
* Feature Importance Graph
* Classification Report
* Test Predictions
* Risk Distribution Chart

## Best Model

Random Forest

## Accuracy

100%

## How to Run

```bash
python3 disease_prediction.py
```
